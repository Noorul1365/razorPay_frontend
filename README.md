# 🛍️ E-Commerce Frontend with Razorpay Payment

A complete e-commerce frontend built with React, Vite, and Razorpay payment integration.

## ✨ Features

- 🔐 User authentication (Login/Register)
- 🛒 Add to cart functionality
- 📦 Product listing page
- 🏠 Checkout with address form
- 💳 Razorpay payment integration
- 📱 Responsive design
- 🔄 Automatic cart management

## 🛠️ Tech Stack

React, Vite, React Router DOM, Axios, Razorpay SDK

## 📋 Prerequisites

- Node.js (v14+)
- Backend API running (Razorpay backend)

## 🚀 Installation

```bash
git clone https://github.com/Noorul1365/razorPay_frontend.git
cd razorPay_frontend
npm install

Create .env file:

VITE_API_URL=http://localhost:500
VITE_RAZORPAY_KEY=rzp_test_xxxxxxxxxxxx

Start development server:

npm run dev
